# ferfereh

Ferfereh is a [3d-printed piece](./3d/gen6-c4.stl) of graffiti with an auto-generated map: [geojson](./coords.geojson), [qgiscloud.com](https://qgiscloud.com/kamangir/ferfereh-published).

# mac
ferfereh help
